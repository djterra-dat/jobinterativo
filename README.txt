# Job Interativo - Site Oficial

## Estrutura
- index.html → Página principal do site
- /assets/ → Pasta para inserir sua logo e ícones

## Publicação

### Opção 1: Hospedagem gratuita (GitHub Pages)
1. Crie uma conta no GitHub.
2. Crie um novo repositório chamado `jobinterativo`.
3. Faça upload do arquivo `index.html` e da pasta `assets`.
4. Vá em *Settings → Pages* e selecione a branch principal como fonte.
5. O site ficará disponível em `https://seuusuario.github.io/jobinterativo`.

### Opção 2: Hospedagem no seu domínio
1. Acesse o painel do seu domínio (por exemplo, HostGator, Locaweb, etc.).
2. Envie o arquivo `index.html` e a pasta `assets` para a pasta pública (geralmente `public_html` ou `www`).
3. Pronto — o site estará acessível em `https://www.jobinterativo.com.br`.
